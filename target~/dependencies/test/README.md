# Overview
MicroEJ Java library API: junit.

# Usage
Add the following line to your `module.ivy` or your `ivy.xml`:
> `<dependency org="ej.junit" name="junit" rev="1.+"/>`

# Requirements
  - EDC-1.2 or higher

# Dependencies
_All dependencies are retrieved transitively by Ivy resolver_.
